# AutoRanker

AutoRanker is a lightweight addon for **World of Warcraft 1.12.1** that automatically updates lower-rank spells on your action bars when you learn a higher rank.

## Features

* Automatically updates outdated spell ranks after learning new spells
* Manual update command with `/arank`
* Scans your action bars and replaces lower-rank spells with the highest rank you know
* Lightweight and only runs when needed
* Made for **WoW 1.12.1 / Vanilla clients**

## Installation

1. Download the addon.
2. Place the `AutoRanker` folder inside your WoW AddOns folder:

```text
World of Warcraft/Interface/AddOns/AutoRanker
```

## How to Use

AutoRanker should run automatically after you learn a new spell rank from your trainer.

You can also update your action bars manually by typing:

```text
/arank
```

The addon will scan your action bars and update any lower-rank spells it finds.

## Notes

* This addon is intended for **World of Warcraft 1.12.1**.
* It is designed for normal ranked spells placed directly on action bars.
* It does not replace macros.
* Some players may intentionally use lower-rank spells for mana efficiency or special situations. In that case, you may want to place those spells manually after using `/arank`.

## Slash Command

```text
/arank
```

Scans your action bars and updates outdated spell ranks.
